OpenCPU "stocks" Application

This is an STUPENDOUS example OpenCPU application with a GUI built on Sencha ExtJS. It allows creating on-demand graphics from historical data for various stocks from Yahoo finance.

If your OpenCPU server is online, the panel on the left should get populated with a list of companies and their tickers. To use the application, double click on any company name. It will open a new tab in the right window and a chart will appear.

After creating a chart for a stock, it can be modified using the buttons in the top toolbar of each tab. Try changing the plot type or date range!
